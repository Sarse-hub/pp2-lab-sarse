# Python Learning Examples - Practice03

This folder contains comprehensive Python learning materials organized into four main categories: Functions, Lambda Expressions, Classes and Objects, and Inheritance. Each directory contains practical examples with explanations.

## Directory Structure

### 📂 functions/
Learn about Python functions - how to define, call, and use them effectively.
- **basic_functions.py** - Function definition and calling
- **function_arguments.py** - Positional and default arguments
- **return_values.py** - Return values and multiple returns
- **args_kwargs.py** - *args and **kwargs

### 📂 lambda/
Master lambda expressions - anonymous functions for concise code.
- **lambda_basics.py** - Lambda syntax and basic usage
- **lambda_with_map.py** - Lambda with map() for transformations
- **lambda_with_filter.py** - Lambda with filter() for selections
- **lambda_with_sorted.py** - Lambda with sorted() for custom sorting

### 📂 classes/
Understand object-oriented programming with classes.
- **class_definition.py** - Class definition and object creation
- **init_method.py** - Constructor (__init__) method
- **class_methods.py** - Instance methods and self parameter
- **class_variables.py** - Class vs instance variables

### 📂 inheritance/
Learn inheritance and polymorphism in Python.
- **inheritance_basics.py** - Parent and child class relationships
- **super_function.py** - Using super() to call parent methods
- **method_overriding.py** - Override parent methods in child classes
- **multiple_inheritance.py** - Inheriting from multiple parent classes

## How to Use

1. Navigate to the appropriate directory for the topic you want to learn
2. Open the Python file in your IDE or text editor
3. Read the comments and docstrings to understand the concepts
4. Run the code to see the output
5. Modify the examples to experiment and deepen your understanding

## Running Examples

To run any Python file:

```bash
python filename.py
```

For example, to learn about basic functions:

```bash
cd functions
python basic_functions.py
```

## Learning Path

**Beginner:** Start with functions/ directory to understand the basics of functions.

**Intermediate:** Move to lambda/ to learn functional programming concepts.

**Advanced:** Study classes/ and inheritance/ to master object-oriented programming.

## Key Concepts Covered

- Function definition and calling
- Function arguments (positional, default, *args, **kwargs)
- Return values and multiple returns
- Lambda expressions and their uses
- map(), filter(), and sorted() with lambdas
- Class definition and instantiation
- Constructors and initialization
- Instance and class variables
- Methods and self parameter
- Inheritance and code reuse
- Method overriding and polymorphism
- super() function
- Multiple inheritance
