# Practice6 - Python File Handling and Built-in Functions

## Structure
- `file_handling/`
  - `read_files.py`
  - `write_files.py`
  - `copy_delete_files.py`
- `directory_management/`
  - `create_list_dirs.py`
  - `move_files.py`
- `builtin_functions/`
  - `map_filter_reduce.py`
  - `enumerate_zip_examples.py`

## How to run
1. `cd c:\PP2\Practice6\file_handling`
2. `python write_files.py`
3. `python read_files.py`
4. `python copy_delete_files.py`

Then run `directory_management` and `builtin_functions` scripts similarly.

## Description
All scripts show usage of file modes (r/w/a), read and write operations, directory creation/listing, file move/copy/delete, built-in functions `map`, `filter`, `reduce`, `enumerate`, `zip`, `len`, `sum`, `min`, `max`, and type conversion.
